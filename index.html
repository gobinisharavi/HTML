📗Tech Glossary

1.HTML
HTML stands for HyperText Markup Language. It is the basic language used to create and structure webpages. HTML uses different tags to define elements like headings, paragraphs, images, links, and videos. It forms the skeleton of a website and tells the browser how the content should appear on the webpage.
2.CSS
Cascading Style Sheets.It is used to design and style webpages created with HTML. CSS helps to change colors, fonts, layouts, spacing, and backgrounds of a webpage. It makes the website look attractive and improves the overall appearance of the webpage.
3.JS
Javascript is a programming language used to make webpages interactive and dynamic. It can control webpage behavior, respond to user actions, and update content without reloading the page. JavaScript is commonly used for features like form validation, animations, pop-ups, and interactive elements.
